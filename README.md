# Power BI Dashboard Project

# Overview

This project contains a Power BI dashboard created to analyze and visualize data in an interactive way. The dashboard helps in understanding key insights, trends, and performance metrics.

# File Information

- File Name: Practical test.pbix
- Tool Used: Power BI Desktop

# Features

- Interactive dashboard
- Data visualization using charts and graphs
- Key performance indicators (KPIs)
- Easy-to-understand layout
